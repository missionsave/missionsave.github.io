MissionSave is exploring new ways to approach sustainable robotics and agriculture. We are developing a fully open-source 3D design tool that aims to combine ideas from well-known tools like AutoCAD, FreeCAD, and Inventor — while keeping it lighter, faster, and parametric.

From greenhouse frames to robotic components, the goal is to make engineering a bit more accessible, accurate, and adaptable for anyone interested in contributing or experimenting.

MissionSave is an open-source initiative — the code is available here: 🔗 https://github.com/missionsave/missionsave.github.io

This tool is being created with the MissionSave robot and greenhouse structure in mind, but we hope it can be useful more broadly. We’re working toward a balance of simplicity, precision, and ease of use, with the flexibility to support different design needs — from structural frames and robotic parts to greenhouse layouts.

This tool is designed to work hand in hand with the Frobot driver, which controls the robot. The driver is written in c++ and uses the Lua programming language, making it easier to fine-tune movements and customize tasks for different applications.
